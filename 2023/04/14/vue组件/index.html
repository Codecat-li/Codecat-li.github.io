VUE组件的基本结构

2023-04-14 15:31:44

vue组件

组件可以解决传统开发中文件依赖不容易维护，可能发生冲突的问题。同时可以增加代码的复用率。

组件是用来实现局部（特定）功能的代码集合。

实际开发中由vm管理app组件，app组件嵌套其他所有组件。

非单文件组件

<body>
  <div id="root">
    <school></school>
  </div>
</body>
<script>
  const school = Vue.extend({
    template: `
      <div>
        <h2>学校名称：{{schoolName}}</h2>
        <h2>学校地址：{{address}}</h2>
      </div>`,
    data() {
      return {
        schoolName: '上鬼狐',
        address: '北京昌平'
      }}})
  new Vue({
    el: '#root',
    components: {
      school,
      student
    }})
</script>

VueComponent构造函数

组件本质是一个VueComponent构造函数，Vue解析的时候会帮我们创建组件同名的构造函数。每次调用Vue.extend返回的都是一个全新的VueComponent。

关于this的指向

  • 组件配置中：data、methods、watch、computed中的函数均指向【VueComponent】

  • new Vue()中：data、methods、watch、computed中的函数均指向【Vue实例对象（vm）】

  • VueComponent有的功能vm都有，但是vm可以绑定el。

  • VueComponent.prototype.proto===Vue.prototype

单文件组件

一个单文件组件包含一下结构：

<template>
  <div>   
  </div>
</template>
<script>
    import 其他组件 from 'url'//不必要
    export default{
        name:'组件名称',
        components:{},//不必要
        data(){
            retun{}
        },
        methods:{}
    }
</script>
<style></style>

所有的组件都遵从vm的管理，我们用main.js来管理

import App from 'url'
new Vue({
el:'#容器id',
 components:{APP}
})

我们还需要一个index.html文件来做容器

<!DOCTYPE html>
<html lang="en">

<head>
  <meta charset="UTF-8">
  <title>只显示了基本结构，删除了无关代码</title>
  <script src="../vue.js"></script>
  <script src="./main.js"></script>
</head>

<body>
    <!--容器-->
  <div id="root">
    <school></school>
  </div>
</body>
