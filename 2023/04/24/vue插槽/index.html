vue插槽

作用：让父组件可以向子组件指定位置插入html结构，也是一种组件间通信的方式，适用于父组件到子组件。

分类如下：

默认插槽

父组件：
<子组件名称>
<div>html结构</div>    
</子组件名称>

子组件：
插槽等着组件的使用者进行填充
<slot>可以写一个默认值，当使用者没有传递具体结构时出现</slot>

具名插槽

父组件：
<div slot='center'>
    <img>
</div>
//v-slot只能用在模板标签里
<template V-slot:center>
    <img>
</template>
<template slot='center'>
    <img>
</template>
------------
子组件：
<slot name = center></slot>可以写一个默认值，当使用者没有传递具体结构时出现</slot>

作用域插槽

数据在插槽组件自身，数据的呈现形式则由插槽使用者来决定。

<Category>
	<template slot-scope="scopeData">
    	<ul>
       		<li v-for="g in scopeData.games" :key="g"></li>
    	</ul>
	</template>

//{{解构赋值}}，否则要加上组件名
	<template slot-scope="{{games}}">
     	<ol>
        	<li v-for="g in scopeData.games" :key="g"></li>
    	</ol>
	</template>
</Category>
------------
<template>
	<div>
    	<slot :games ="games"></slot>默认值</slot>
	</div>
</template>

<script>
export default{
    name:"Category",
    props:['title'],
    data(){
return{
games:['q','w']
		}
    },
}
</script>
