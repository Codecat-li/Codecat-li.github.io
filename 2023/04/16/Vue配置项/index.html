vue配置项

props 配置项

​ 功能：让组件接收外部传来的数据。props是只读的，如果进行修改，会收到警告，可以把props复制一份到data中，然后修改data。

1.传递数据：
<Demo name="xxx"/>
2.接收数据：
props:['name']//只接收数据
props:{name:Number}//限制接收数据的类型
props:{
	name:{
	type:String,//类型
	required:true,//必要性
	default:'老王'//默认值
	}
}
mixmin 混入

​ 功能：可以把多个组件共用的配置提取成一个混入对象

​ 使用方式:

1.定义混合：
export const xxx ={
data(){...},
methods:{...},
...
}
2.使用混入：
使用import引入文件中然后：
(1)全局混入：vue.mixin(xxx)
(2)局部混入：mixins:['xxx']
plugins 插件

功能：用于增强vue，本质是包含install方法的对象，install的第一个参数是vue,第二个参数是使用者传递的数据。

1.定义插件：
对象.install= function(Vue,options){
Vue.filter(...)
Vue.directive(...)
Vue.mixin(...)
}
2.引用插件：
VUe,use()
scoped样式属性

功能：让样式在局部生效防止冲突

<style scoped>
